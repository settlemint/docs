## Create Private Key

<Tabs items={['Platform UI', 'SDK CLI', 'SDK JS']}>
  <Tab value="Platform UI">
    <Steps>
      ### Navigate to Application
      Go to your application

      ### Access Private Keys
      Click **Private keys** in the left navigation

      ### Configure Key
      1. Click **Create a private key**
      2. Choose key type
      3. Enter name
      4. Select nodes
      5. Click **Confirm**
    </Steps>
  </Tab>

  <Tab value="SDK CLI">
    ```bash
    # Create Accessible ECDSA P256 key
    settlemint platform create private-key accessible-ecdsa-p256 my-key \
      --application my-app \
      --blockchain-node node-123

    # Create HD ECDSA P256 key
    settlemint platform create private-key hd-ecdsa-p256 my-key \
      --application my-app

    # Create HSM ECDSA P256 key
    settlemint platform create private-key hsm-ecdsa-p256 my-key \
      --application my-app
    ```
  </Tab>

  <Tab value="SDK JS">
    ```typescript
    const createKey = async () => {
      const result = await client.privateKey.create({
        name: "my-key",
        applicationUniqueName: "my-app",
        privateKeyType: "ACCESSIBLE_ECDSA_P256",
        blockchainNodeUniqueNames: ["node-123"]
      });
    };
    ```
  </Tab>
</Tabs>

## Manage Private Keys

<Tabs items={['Platform UI', 'SDK CLI', 'SDK JS']}>
  <Tab value="Platform UI">
    Navigate to your private key to:
    - View details and status
    - Manage node associations
    - Check balances
    - Fund the key
  </Tab>

  <Tab value="SDK CLI">
    ```bash
    # List keys
    settlemint platform list private-keys --application <app-name>

    # Get details
    settlemint platform read private-key <name>

    # Restart key
    settlemint platform restart private-key <name>
    ```
  </Tab>

  <Tab value="SDK JS">
    ```typescript
    // List keys
    await client.privateKey.list("your-app");

    // Get details
    await client.privateKey.read("key-name");

    // Restart key
    await client.privateKey.restart("key-name");
    ```
  </Tab>
</Tabs>

## Fund Private Key

<Steps>
  ### Access Key Details
  Click the private key in the overview

  ### Open Balances
  Navigate to the **Balances tab**

  ### Add Funds
  1. Click **Fund**
  2. Scan QR code with wallet/exchange
  3. Complete transaction
</Steps>

<Callout type="warning">
Ensure your private key has sufficient funds before attempting transactions on networks that require gas fees.
</Callout>
