## Add Node

<Tabs items={['Platform UI', 'SDK CLI', 'SDK JS']}>
  <Tab value="Platform UI">
    <Steps>
      ### Navigate to Application
      Go to the application containing your network

      ### Access Nodes
      Click **Blockchain nodes** in the left navigation

      ### Configure Node
      1. Click **Add a blockchain node**
      2. Select the blockchain network
      3. Choose node name and type
      4. Configure deployment settings
      5. Click **Confirm**
    </Steps>
  </Tab>

  <Tab value="SDK CLI">
    ```bash
    # Create node
    settlemint platform create node besu <name> \
      --network <network-name> \
      --type <VALIDATOR|NON_VALIDATOR>

    # Get help
    settlemint platform create node --help
    ```
  </Tab>

  <Tab value="SDK JS">
    ```typescript
    const createNode = async () => {
      const result = await client.node.create({
        name: "my-node",
        networkUniqueName: "my-network",
        nodeType: "VALIDATOR",
        provider: "provider",
        region: "region"
      });
      console.log('Node created:', result);
    };
    ```
  </Tab>
</Tabs>

## Manage Node

<Tabs items={['Platform UI', 'SDK CLI', 'SDK JS']}>
  <Tab value="Platform UI">
    Navigate to your node and click **Manage node** to:
    - View node details and status
    - Monitor node health
    - Restart node operations
  </Tab>

  <Tab value="SDK CLI">
    ```bash
    # List nodes
    settlemint platform list nodes --application <app-name>

    # Restart node
    settlemint platform restart node <name>
    ```
  </Tab>

  <Tab value="SDK JS">
    ```typescript
    // List nodes
    await client.node.list("your-app");

    // Get node details
    await client.node.read("node-name");

    // Restart node
    await client.node.restart("node-name");
    ```
  </Tab>
</Tabs>

<Callout type="info">
All operations require appropriate permissions in your workspace.
</Callout>
