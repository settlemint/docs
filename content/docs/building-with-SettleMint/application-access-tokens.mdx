## Create Access Token

<Steps>
  ### Navigate to Tokens
  Go to your application's dashboard and click "App access tokens"

  ### Configure Token
  1. Click **Add an application access token**
  2. Choose token name
  3. Set expiration date
  4. Select scope type:
     - **All** - Access to all services
     - **Specific** - Select individual services

  ### Generate Token
  Click **Confirm** to create token
</Steps>

<Callout type="warning">
Copy and save your token securely - you won't see it again. Treat it like a password and keep it secret.
</Callout>

## Update Access Token

<Steps>
  ### Access Token Settings
  1. Navigate to **App Access Tokens**
  2. Click **View scopes** for your token

  ### Modify Permissions
  1. Click **Update**
  2. Choose new scopes
  3. Click **Confirm**
</Steps>

## Delete Access Token

<Steps>
  ### Navigate to Tokens
  Go to application dashboard's token section

  ### Remove Token
  1. Click **Delete** next to token
  2. Type **DELETE** to confirm
  3. Token becomes invalid immediately
</Steps>

## Using Access Tokens

### Special Cases

For specific services:
- IPFS: `https://myservice.settlemint.com/TOKEN/api/v0/...`
- Avalanche/Fuji: `https://myservice.settlemint.com/ext/bc/C/rpc/TOKEN`
