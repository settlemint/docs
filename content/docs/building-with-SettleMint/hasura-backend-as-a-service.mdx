## Add Hasura

<Tabs items={['Platform UI', 'SDK CLI', 'SDK JS']}>
  <Tab value="Platform UI">
    <Steps>
      ### Navigate to Application
      Go to your application

      ### Access Integration Tools
      Click **Integration tools** in the left navigation

      ### Configure Hasura
      1. Click **Add an integration tool**
      2. Select **Hasura**
      3. Enter name and configure settings
      4. Click **Confirm**
    </Steps>
  </Tab>

  <Tab value="SDK CLI">
    ```bash
    # Create Hasura instance
    settlemint platform create integration-tool hasura <name>

    # Get help
    settlemint platform create integration-tool hasura --help
    ```
  </Tab>

  <Tab value="SDK JS">
    ```typescript
    const createHasura = async () => {
      const result = await client.hasura.create({
        name: "my-hasura",
        applicationUniqueName: "my-app"
      });
    };
    ```
  </Tab>
</Tabs>
