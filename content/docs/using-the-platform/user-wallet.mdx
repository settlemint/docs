## Setup User Wallets

<Steps>
  ### Navigate to Application
  Go to your application's private keys section

  ### Create HD Key
  1. Click **Create a private key**
  2. Select **HD ECDSA P256** type
  3. Enter name
  4. Select nodes/load balancers
  5. Click **Confirm**
</Steps>

![Add User Wallet](../../img/using-the-platform/add-user-wallet.png)

## Create User Wallets

<Steps>
  ### Access Wallet Details
  When status is **Running**, click to view details

  ### View Key Material
  Check the Mnemonic under **Key material**

  ### Add User Wallet
  1. Click **Create a user wallet**
  2. Enter wallet name
  3. Click **Confirm**
</Steps>

![Create User Wallet Form](../../img/using-the-platform/create-user-wallet-form.png)

<Callout type="info">
The new user wallet appears in the list with its associated address. For networks requiring gas, ensure the wallet contains sufficient funds.
</Callout>
