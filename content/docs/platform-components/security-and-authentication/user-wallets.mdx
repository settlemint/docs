## Enterprise User Wallet Setup

<Callout type="success" title="Enterprise Deployment">
**Production-Ready**: Configure enterprise user wallets with HD ECDSA P256 security for unlimited scalability and enhanced user privacy across your blockchain infrastructure.
</Callout>

### Initial Configuration Process

<Steps>

### Access Private Key Management
Navigate to your enterprise application and select **Private keys** from the left navigation panel to access enterprise key management interface.

### Initialize User Wallet Creation
Click **"Create a private key"** to open the enterprise key configuration form with advanced wallet deployment options.

### Configure HD Wallet Parameters
- **Private Key Type**: Select **HD ECDSA P256** for hierarchical deterministic wallet architecture
- **Deployment Name**: Enter a descriptive name for enterprise tracking and management
- **Node Selection**: Choose target nodes or load balancers for user wallet deployment (modifiable post-deployment)

### Deploy Enterprise Infrastructure
Click **Confirm** to deploy your enterprise user wallet infrastructure with production-ready HD security.

</Steps>

## Enterprise User Wallet Creation

### HD Wallet Infrastructure Management

<Callout type="info" title="Master Key Architecture">
**Secure Foundation**: Enterprise user wallets derive from a cryptographically secure mnemonic seed, enabling unlimited wallet generation with simplified backup and recovery procedures.
</Callout>

<Steps>

### Verify Deployment Status
Monitor your deployment until status shows **Running**, then click to access detailed enterprise wallet infrastructure information.

### Review Master Key Material
Access the **Key material** section to view the hierarchical deterministic mnemonic from which all enterprise user wallets are securely generated.

### Access User Wallet Management
Navigate to the **User Wallets** section (initially empty) and click **"Create a user wallet"** to begin individual wallet provisioning.

### Configure Individual Wallet
Complete the wallet creation form by entering a descriptive **wallet name** for enterprise user identification and tracking.

### Verify Wallet Deployment
Review the newly created user wallet in the enterprise management list, displaying the unique blockchain address for transaction operations.

</Steps>
