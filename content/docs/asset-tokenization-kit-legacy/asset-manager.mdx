## 1. Details Tab: What key metrics should I monitor?

The Details tab provides critical real-time information that compliance officers, treasurers, and asset managers need to monitor:

### Essential Asset Identifiers
- **Asset Name & Trading Symbol**: Your custom branding (e.g., "Euro Stablecoin" with symbol "EURS")
- **Smart Contract Address**: Immutable blockchain identifier for integration with wallets and exchanges
- **Asset Creator**: Original issuer's verified wallet address for transparency
- **Decimal Precision**: Token divisibility (18 decimals = standard, like ETH)

### Real-time Supply Metrics
- **Total Supply**: Current circulating tokens (updates instantly with mint/burn operations)
- **Burned Tokens**: Permanently removed from circulation for supply management
- **Market Capitalization**: Total Supply √ó Unit Price (real-time valuation)
- **Daily Volume**: 24-hour transaction volume indicating liquidity

### Holder Analytics
- **Total Holders**: Number of unique wallets (indicates adoption rate)
- **Ownership Concentration**: Top holder's percentage (regulatory metric for market manipulation risk)
  - **< 10%**: Highly distributed (ideal for public tokens)
  - **10-50%**: Moderate concentration (typical for corporate tokens)
  - **> 50%**: High concentration (common during initial issuance)

### Pricing Information
- **Unit Price**: Current token value in base currency
- **Price History**: 30-day price chart with volatility indicators
- **Market Depth**: Buy/sell order distribution
